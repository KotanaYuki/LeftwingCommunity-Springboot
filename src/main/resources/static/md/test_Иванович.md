# 〔诗〕召唤

*内斯托尔·马赫诺 （1907年）*

起来吧，兄弟们，咱们一齐前进！
人民将会在**黑色**的旗帜下决起，
我们乐于战斗，要给它们通通冲垮，
因为我们信仰公社——这制度对咱忠诚无比。

我们要消灭皇权，把资本家的权力打倒。
我们打碎一切黄金的熔岩，
他们想要尊重，咱们才不要给他。
**他们那卑鄙制度，得用血战来回敬。**

刽子手的锁链，压迫了咱们太久，
给咱们铐上枷锁，关进他的监狱里。
快快列队，伟大斗争的时候到了，
集合的位置就定在的那面黑旗。

再不要给暴君当机器，
要知道，伟大的、力量在咱们手里。
所以，站起来吧，兄弟们——劳动人民，
这群剥削人的害兽，要粉碎得彻底。

***权力和他们那些法令，对咱有什么用？***
***我们的灵魂自由，我们的意志坚定。***
***大小冠冕都打倒，给自由开路！***
***在咱们的世界里，人就该这么活着。***

起来吧，兄弟们，就在那约定之时，
打着黑旗，咱们去暴君那里，
毁灭权力，还有这制度——可耻至极，
正是他们把咱们推向血的战斗。
